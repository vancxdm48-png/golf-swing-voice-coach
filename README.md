# Golf Swing Voice Coach

ゴルフスイングをカメラまたは動画ファイルで確認し、AI姿勢推定、簡易モーション分析、音声読み上げ、履歴比較を使って「次の1球で何を直すか」まで出すブラウザアプリです。

## 主な機能

- カメラ撮影、録画、再撮影
- 手元のスイング動画読み込み
- 背面/前面カメラ切替
- 手動停止、4秒、6秒、8秒の録画時間設定
- 0.5倍再生、1倍再生、1コマ送り、1コマ戻し
- 頭、中心軸/前傾、肩/腰、ボール位置のガイド表示
- MediaPipe Pose LandmarkerによるAI姿勢推定
- AI姿勢推定が使えない環境での簡易モーション解析フォールバック
- 安定度、左右ブレ、テンポ、信頼度の表示
- 診断スコアの根拠表示
- 日本語音声アドバイス
- 次回課題を1つに絞る練習ループ
- 診断履歴のブラウザ内保存
- JSONレポート保存
- 診断文章コピー
- PWA対応、ホーム画面追加、ローカル資産のオフライン起動

## 起動方法

カメラ機能は `file://` では動かないことがあるため、ローカルサーバーで開きます。

```bash
cd /Users/mizutanidaisuke/Documents/Codex/golf-swing-voice-coach
python3 -m http.server 5174
```

ブラウザで次を開きます。

```text
http://localhost:5174
```

## 推奨環境

- Chrome、Edge、Safariの最新版
- スマホでは `localhost`、HTTPS、またはホーム画面追加後のPWA起動
- AI姿勢推定には初回のみMediaPipe関連ファイルの読み込みが必要
- ネットワークが使えない場合は、AI姿勢推定ではなく簡易解析で動作

## 使い方

1. 正面または後方を選ぶ
2. 目標を選ぶ
3. カメラ開始、または動画読込
4. 全身とクラブが画面に入るように撮る
5. 診断を押す
6. 音声アドバイスを聞く
7. 次回課題を1つだけ直して再撮影する
8. 前回履歴と比較する

## プライバシー

このアプリは、動画を外部サーバーへアップロードしません。録画、動画解析、履歴保存はブラウザ内で行います。

履歴はブラウザの `localStorage` に保存します。動画本体は保存しません。

## 注意

この診断は練習用の目安です。医療診断、身体機能評価、専門コーチングの代替ではありません。

撮影時は周囲の安全を確認し、クラブを振れる十分なスペースを確保してください。

AI姿勢推定は撮影角度、明るさ、服装、背景、クラブの見え方で精度が変わります。結果は断定ではなく「確認ポイント」として扱ってください。
